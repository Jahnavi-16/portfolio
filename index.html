Family Travel Tracker

The Family Travel Tracker project is an interactive web application that features a personalized map for each family member. Each member can select their own color to mark the places they've visited, and new members can be added with ease. Secrets app(Similar to whispers)

The Secrets project is a web application that allows users to anonymously share and store their secrets securely. The app features Google OAuth 2.0 for authentication, ensuring that only authorized users can access the platform. It uses hashing to protect passwords and Axios for seamless API communication. Once logged in, users can perform full CRUD operations on their secrets, all of which are stored in a PostgreSQL database. Though the interface is simple, the application effectively balances user privacy with functionality.
